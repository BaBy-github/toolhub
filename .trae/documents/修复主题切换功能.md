## 问题分析
1. 项目使用 Tailwind CSS v4，但配置文件中仍然使用了 v3 的 `darkMode: 'class'` 配置
2. 在 Tailwind CSS v4 中，`darkMode` 配置已被移除，默认支持 class 模式的暗色主题切换
3. 主题切换的逻辑代码是正确的，但由于配置不兼容，dark 类没有被正确应用

## 解决方案
1. 修改 `tailwind.config.js` 文件，移除不兼容的 `darkMode` 配置
2. 确保 Tailwind CSS v4 正确应用 dark 类
3. 测试主题切换功能
