# 实现对称加密工具

## 1. 功能需求
参考ToEscape工具，实现一个对称加密工具，支持：
- 加密和解密模式切换
- 多种对称加密算法（如AES-256-CBC）
- 自定义密钥和初始化向量（IV）
- 生成随机密钥和IV
- 实时转换

## 2. 实现步骤

### 2.1 创建加密工具实现文件
- 创建 `src/utils/crypto.ts`，实现对称加密和解密功能
- 支持AES-256-CBC算法
- 提供加密、解密、生成随机密钥和IV的函数

### 2.2 创建加密工具页面组件
- 创建 `src/pages/ToCrypto.vue`，参考ToEscape组件的结构
- 实现输入输出区域
- 添加加密/解密模式切换
- 添加算法选择和密钥设置
- 实现随机密钥和IV生成功能

### 2.3 更新工具列表配置
- 修改 `src/data/tools.ts`，添加对称加密工具配置

### 2.4 添加国际化支持
- 更新 `src/i18n/locales/zh.json`，添加中文翻译
- 更新 `src/i18n/locales/en.json`，添加英文翻译

## 3. 技术实现细节

### 3.1 加密算法实现
- 使用Web Crypto API进行对称加密
- 支持AES-256-CBC算法
- 提供密钥和IV的生成与验证

### 3.2 UI设计
- 参考ToEscape工具的布局和交互
- 左侧输入区域，右侧输出区域
- 支持拖拽调整比例
- 提供设置面板，允许配置算法参数

### 3.3 交互逻辑
- 实时转换：输入变化时自动加密/解密
- 支持模式切换：加密 ↔ 解密
- 支持复制输出结果
- 支持清空输入

## 4. 文件结构
```
src/
├── utils/

├── pages/
│   └── ToCrypto.vue       # 加密工具页面
```
